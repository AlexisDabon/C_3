# EXERCISES BY ORDER IN THIS C FOLDER

I am moving to the next level with this new series of C programming exercises.
I am learning arrays and pointers with this series of short programs for beginners.

* **Exercise 1:** crypto/alert program takes 3 numbers and prints their average. It sends an "ALERT" message if the average is over the limit, otherwise it says the situation is stable.

* **Exercise 2:** crypto/vol second exercise calculates the variation between 3 numbers. If this variation is over 15, it sends an alert to the user.

* **Exercise 3:** crypto/id to work with structures and functions, updating and printing the new price of a cryptocurrency using pointers.
